Dominos API

Unofficial reverse-engineered Dominos Pizza API wrapper for Node.js. Zero dependencies.

Features

  • Store FinderFind nearby stores by address
  • Menu BrowserFull menu, categories, search
  • Order TrackerTrack by phone or order key
  • Order BuilderBuild, validate, price, place
  • Coupon LookupGet coupon details by ID
  • US + Canadadominos.com and dominos.ca

Quick Start

import { DominosAPI } from './dominos.js';

const api = new DominosAPI({ region: 'ca' });

// Find stores
const stores = await api.stores.find('20690 40 Ave, Langley, BC');

// Browse menu
const menu = await api.menu.get(stores[0].StoreID);

// Track an order
const status = await api.tracker.byPhone('6045342277');
