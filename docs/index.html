Getting Started

Each bot lives in its own directory with full documentation. Clone the repo and explore: 

git clone https://github.com/nulljosh/bots.git
cd bots/foodie
npm install
node -c src/chipotle.js  # Verify syntax

# Or use OpenClaw commands:
/food chipotle bowl:chicken guac
